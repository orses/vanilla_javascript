# ENGLISH WORD DEFINITIONS

A project to consume data from an external REST API, using Fetch API and async/await

## How to use this repository

- To install

  ```js
  yarn install
  ```

- To run the development server

  ```js
  yarn dev
  ```

- To build a production website

  ```js
  yarn run build
  ```
